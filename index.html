What agents can say when they have real data
"Find me a action game I can play in short sessions — something with a real challenge to it."
Without Barcode
Agent response:
  • —God of War — great action game
  • —Assassin's Creed has similar elements
  • —The Last of Us is a top pick

Decent genre match. Session length wasn't a factor — the agent didn't have it to work with.

With Barcode
Agent response:
  • —Hades — 20–30 min runs, high replayability
  • —Dead Cells — pick up and put down, no checkpoints needed
  • —Returnal — intense sessions, built around short loops

Session length is a structured field. Filtered, not guessed.

Also available with Barcode data
Hardware check "Looks like your machine can handle it — your GPU clears the recommended spec with room to spare."
Live server signal "Servers are packed right now — player count is peaking. Best time to find a squad."
Library awareness "You already own Hades — you'll feel right at home. The dodge timing and loop structure are nearly identical."

The difference isn't a smarter model. It's structured product data the model can actually use.
